Castle Invasion

Base lógica escalonável de um RPG indie.
Arquitetura Data Driven Game Design (DDGD) com Domain Respect (DR) como bússola.

Em suma, em termos de importação e semântica:

data -> NADA
core -> data | stdlib
entities -> core | stdlib 
systems -> entities | core | stdlib
ui -> systems | core
main -> TUDO

OBS.: entities/ e ui/ acessam data/ por meio de core/<registry>.py
